# Lyrical Studio Pro

这是一个专为音乐爱好者和歌词校对者设计的专业级 MP3 元数据与歌词同步工具。它允许用户在浏览器中直接编辑 MP3 标签、提取/保留专辑封面，并通过实时“敲击节奏”的方式精准对齐歌词时间轴。

## 🌟 核心功能

- **元数据管理**：自动读取并显示 MP3 的标题、艺术家、专辑及内嵌高清封面。
- **歌词内嵌**：支持读取和写入 ID3v2 USLT (Unsynchronised lyrics) 帧。
- **节奏同步引擎**：
  - **全曲同步**：从头开始随音乐节奏敲击 `Enter` 键记录时间。
  - **断点续传同步**：支持从指定歌词行开始重新打点，无需重头再来。
- **纯文本导入**：支持直接粘贴纯文本歌词，自动分行并过滤空行。
- **导出与封装**：保存时将所有修改（包括原始封面和新歌词）重新封装进 MP3 文件，无损质量。
- **响应式 UI**：采用现代沉浸式深色模式设计，适配各种屏幕尺寸。

## 🛠 技术栈

- **React 19**：构建用户界面。
- **Tailwind CSS**：实现高度定制化的玻璃拟态（Glassmorphism）视觉效果。
- **jsmediatags**：用于高性能的客户端 MP3 标签读取。
- **browser-id3-writer**：在浏览器端构建和写入 ID3v2 二进制结构。
- **Lucide-like Icons**：使用精美的 SVG 图标增强视觉表现。

## 📖 工作原理

本项目深入利用了 HTML5 Audio API 和二进制数据处理技术。详细的实现细节和教学文档请参阅项目中的 `principles.html` 文件。

### 关键流程：
1. **读取**：File API -> ArrayBuffer -> jsmediatags 解析。
2. **同步**：Audio currentTime -> React State 映射 -> 自动滚动。
3. **写入**：ID3v2 帧封装 -> Blob 生成 -> 模拟下载。

## 🚀 快速开始

1. 上传一个 MP3 文件。
2. 导入歌词文本（或直接从文件读取）。
3. 点击“全曲打点”或在特定行点击“从此处开始打点”。
4. 听着音乐，在每句歌词开始时按下回车。
5. 点击“导出同步文件”获取成品。
